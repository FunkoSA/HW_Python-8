# Проект телефонный справочник
## Призводимые действия:
 * Поиск записей
 * Вывод записей
 * Импорт записей в справочник
 * Экспорт записей из справочника

## Реализация импорта-экспорта:
 * файлы JSON
 * файлы CSV
## Дополнительно реализована генерация списков контактов для импорта
